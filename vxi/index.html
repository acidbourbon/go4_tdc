This page allows to access any lab equipment that runs the VXI-11 protocol (such as (Tektronix|Agilent) (Scopes|SignalGenerators|MultiMeters)). The lib is C-based, so it needs to be recompiled if the available x64-binary does not work. The code was mainly downloaded from [VXI-11 library], but small changes were applied to the tool. The source code is included in the repository. Enter the IP of the device and any command and you'll receive the answer.
